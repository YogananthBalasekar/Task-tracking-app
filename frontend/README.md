# Project Setup

This project contains both **Backend (Node.js)** and **Frontend (React + Vite)** applications.

Follow the steps below to run the project locally.

---

## 1. Run the Backend

Open a terminal and run:

cd backend
npm install
npm start
2. Run the Frontend (Development)

Open another terminal and run:

cd frontend
npm install
npm run dev

After running this command, the application will start and open in your browser.

Example:

http://localhost:5173

3. Build the Frontend

To create a production build of the frontend:

npm run build

4. Preview the Production Build

To preview the built application:

npm run preview
